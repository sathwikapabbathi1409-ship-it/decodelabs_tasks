**task 1.decodelabs_chatbot.py:**
What is a Rule-Based Chatbot?
A chatbot that replies based on predefined rules
Uses if-else conditions / pattern matching
No machine learning involved
Best for simple FAQs and basic conversations
* Key Components
User Input → text entered by user
Rules/Patterns → keywords or phrases to match
Responses → predefined replies
Logic Engine → if-else or dictionary mapping
* How It Works
User enters a message
Program checks for keywords
Matches rule
Returns corresponding response
* Simple Python Approach
Use input() for user message
Use if-elif-else or dictionary
Use while loop for continuous chat
Use lower() to normalize text
